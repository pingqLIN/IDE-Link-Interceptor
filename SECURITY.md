## Security Policy

## About This Project

This project is a personal interest development. I will make every effort to maintain code integrity and security.

## Reporting Security Vulnerabilities

If you discover a security vulnerability or security-related issue, please contact me via the following methods:

- **Do not** publicly disclose security issues
- Report privately via GitHub Issues or directly contact the project maintainer
- Provide a detailed vulnerability description and steps to reproduce

## Security Updates

Although this is a personal project, I will:

- Respond to security reports promptly to the best of my ability
- Fix known security issues within my capabilities
- Release new versions for critical security updates after resolution
